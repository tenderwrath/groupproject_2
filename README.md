## Оценка влияния продвижения товаров бренда в VK на их спрос на Wildberries

Бренд ведёт сообщество VK и продаёт товары на WB. Ему важно понять, что действительно ли упоминания товара в соцсети связаны со спросом на маркетплейсе, есть ли признаки того, что товары с более активной поддержкой в VK лучше продаются / чаще получают отзывы / имеют больше заказов и стоит ли вообще стратить средства на продвижение товаров в VK

Наша основная гипотеза заключается в том, что чем чаще и активнее конкретный товар упоминается в сообществе VK, тем выше его популярность на WB

## Используемые скрипты
В папке [scripts](./scripts) находятся все парсеры, такие как:

[vk_api_searched](./scripts/vk_api_searched.py) - код API, который с помощью метода `wall.search` получает 100 последних постов с каждого паблика VK рассматриваемых брендов, в которых упоминается WB

[vk_api_wall_posts](./scripts/vk_api_wall_posts.py) - код API, который с помощью метода `wall.get` просто получает 100 последних постов с каждого паблика VK рассматриваемых брендов

[wb_parser](./scripts/wb_parser.py) - парсер, который с помощью библиотеки `selenium` по каждому бренду парсит страницу WB до самого конца

[wb_links_parser](./scripts/wb_links_parser.py) - парсер, который из прямых ссылок на WB получает артикулы товаров через перехват редиректов

## Используемые библиотеки

В проекте были использованы следующие библиотеки:

- **time** для работы со временем
- **csv** чтение и запись данных в CSV-файлы
- **logging** логирование
- **re** регулярные выражения для поиска и обработки текста
- **os** работа с файлами и путями
- **json** работа с JSON-данными
- **requests** отправка HTTP-запросов в API
- **selenium** автоматизация браузера для парсинга
- **pandas** для работы с данными
- **matplotlib** графики
- **seaborn** графики
